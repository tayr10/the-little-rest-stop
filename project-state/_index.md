# The Little Rest Stop — Website Project

**What is this?**
A marketing website for The Little Rest Stop, a family-run breakfast and lunch diner
in Sturbridge, MA. This is Taygan's mom's business.

The goal is a live, customer-ready website at a real domain that loads fast, works on
mobile, and helps new customers find the restaurant, see the menu, and get directions.

**Stack:** Plain HTML + CSS + JS. No frameworks. No build tools.
**Hosting target:** Netlify (free) + custom domain (TBD)
